# RestaurantAIBot Restaurant Marketing Plan

Last updated: 2026-05-28

## Purpose

RestaurantAIBot is a worldwide AI restaurant discovery app. The visitor side helps people search for restaurants by craving, city, travel destination, mood, budget, dining style, or nearby location.

The business side will help restaurants and local advertisers get in front of people already searching for food.

## Important status

This plan is not live ad code.

Do not add payments, live ad scripts, campaign tracking, dashboards, sponsored placements, or private keys until those items are intentionally approved.

## Future product offers

### 1. Restaurant profile page

A profile page can include:

- restaurant name
- city and neighborhood
- cuisine
- description
- menu highlights
- photos or image placeholders
- website link
- phone link
- map link
- offer text
- disclosure if paid

### 2. Promo copy builder

Input:

- restaurant name
- city
- cuisine
- offer
- target customer
- preferred tone

Outputs:

- website promo copy
- social media caption
- email promo
- flyer headline
- Google Business Profile post idea
- short call-to-action text

### 3. Local visibility package

Possible placements after disclosure setup:

- city page mention
- cuisine page mention
- search-related featured card
- popular searches page mention
- restaurant profile link

All paid placements must be clearly labeled.

### 4. Canva-ready design kit

Create text and layout direction for:

- square social post
- story/reel cover
- flyer
- coupon card
- table tent copy
- QR code call-to-action

### 5. Monthly refresh package

Potential monthly service:

- one updated offer
- one refreshed profile section
- four social captions
- one flyer concept
- one city/cuisine visibility review

## Starter package ideas

### Starter Listing

For restaurants that only need a basic presence.

Includes:

- profile page draft
- basic description
- map/contact links
- simple call-to-action

### Local Promo Kit

For restaurants promoting a special.

Includes:

- offer copy
- social caption
- flyer headline
- call-to-action

### City Visibility Interest

For restaurants wanting future visibility on city or cuisine pages.

Includes:

- intake review
- city/cuisine match
- disclosure-ready placement plan
- no live placement until approved
