# SEO-GEO-STANDARD.md

## Purpose

Use this formula on important pages:

SEO foundation + GEO answer structure + AEO question formatting + agent files + clean GitHub instructions + crawler checks.

## Page rules

Every important page should:

1. Answer the visitor's question quickly.
2. Use one clear H1.
3. Use helpful H2 sections.
4. Include short question-and-answer sections when useful.
5. Use simple language.
6. Link to related pages.
7. Have accurate title and meta description.
8. Have a canonical URL when appropriate.
9. Be included in the sitemap when public and useful.
10. Be listed in `public/llms.txt` when it is an important evergreen page.

## Site rule

RestaurantAIBot pages should help locals and travelers find useful restaurant choices by location, meal, place, or occasion.

Keep the site plain HTML, CSS, and JavaScript. Do not redesign or change architecture unless Gerry asks.
