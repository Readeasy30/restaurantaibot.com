FAQ

How does Restaurant AI Bot work?

You type what you want, add a location, and the app returns a ranked list of restaurant ideas.

Can I save restaurants?

Yes. Use the save button on a result card and the restaurant will appear in your favorites list on this device.

Does it only work for one city?

No. You can search by city or ZIP and use it anywhere your API returns restaurant results.

Yes. The layout is being built so featured listings, city pages, and local sponsorships can be added later.
